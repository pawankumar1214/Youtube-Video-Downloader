
# 🎥 YouTube Video Downloader

A Python-based desktop application that allows users to download YouTube videos and audio with an intuitive graphical interface. The application provides multiple quality options, download progress tracking, and a simple user experience.

## ✨ Features

* 📥 Download YouTube videos
* 🎵 Download audio only (MP3)
* 🎬 Multiple video quality options
* 📊 Live download progress
* 🖥️ User-friendly GUI
* 📂 Save files to a custom location
* ⚡ Fast and reliable downloads

## 🛠️ Technologies Used

* Python
* Tkinter
* yt-dlp
* Pillow (PIL)
* Threading
* OS Module

## 🚀 Installation

### 1. Clone the repository

```bash
git clone git@github.com:pawankumar1214/Youtube-Video-Downloader.git
```

### 2. Navigate to the project folder

```bash
cd Youtube-Video-Downloader
```

### 3. Install the required dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python main.py
```
